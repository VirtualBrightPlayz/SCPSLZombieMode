# SCPSLZombieMode

Suggested by Discord User: Kayckbr

Just drag and drop! No config files yet.

# Recommended settings:

Lower HP of SCPs to balance the game out.

```
scp049_hp: 100
scp096_hp: 200
scp106_hp: 200
scp173_hp: 350
scp939_53_hp: 200
scp939_89_hp: 200
```

## Know that SCP-096 gets *very* buggy when there is multiple.
